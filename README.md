## 📱 About the App

**NutriTrack** is a mobile application designed to help users achieve their health and fitness goals by tracking nutrition and daily activities.  
The app provides an easy-to-use interface with powerful features to maintain a balanced lifestyle.

---

## ✨ Features

- 🥗 **Nutrition Tracking** – Log meals and monitor calories, proteins, carbs, and fats  
- 💪 **Fitness Tracking** – Record workouts and physical activity  
- 📊 **Progress Monitoring** – View charts and stats of your health journey  
- 🎯 **Personalized Goals** – Set daily/weekly health & fitness goals  
- 🌙 **Dark Mode** – Eye-friendly interface  

---

## 🛠️ Tech Stack

- **Frontend:** Flutter (Dart)  
- **Backend:** Firebase / REST APIs (if applicable)  
- **Database:** Cloud Firestore / SQLite
